Este projeto foi desenvolvido como Projeto Integrador Final do programa Bolsa Futuro Digital, uma iniciativa do Governo em parceria com a empresa Aponti e Solutions.

O BFD Inventory é uma solução de front-end moderna desenvolvida em React + Vite, focada em resolver o desafio real de controle de estoque e inventário. O sistema permite o gerenciamento ágil de produtos, com uma interface intuitiva, autenticação de usuários e um design responsivo focado na experiência do usuário (UX).

Através deste protótipo, apliquei conceitos avançados de hooks (useState, useEffect), componentização escalável e estilização customizada com CSS modular, demonstrando a prontidão técnica adquirida durante a formação para atuar no ecossistema de tecnologia.
